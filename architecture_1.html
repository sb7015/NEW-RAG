üìã Citation Link Generation Flow
1. Retrieval: Agent gets chunk from OpenSearch with metadata (s3_path: "hr/policies.pdf", page: 12)
2. URL generation: Lambda generates pre-signed URL. Valid 1 hour. Includes #page=12 anchor
3. Frontend: React renders clickable link "Source: Leave Policy, Page 12"
4. Click: Browser opens PDF viewer directly on page 12 (PDF.js, Chrome PDF viewer, or Adobe)

  • PDF: #page=X
